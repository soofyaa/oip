## Задание 1 (в папке task1)

1. Скачать минимум 100 текстовых страниц с помощью краулера из предварительно подготовленного списка (у нас это urls.txt).
2. Записали текст каждой страницу в директорию task1/vykachka, от html разметки не очищали
3. Создали файл index.txt в котором хранится номер файла и ссылка на страницу
4. Собрали в архив vykachka.zip

Запуск задания:
1. `cd task1`
2. `python main.py`

## Задание 2 (в папке task2)

Запуск задания:
1. `cd task2`

## Задание 3 (в папке task3)
